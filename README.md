# 📈 N100 Finance Intelligence Platform

A Financial Intelligence Platform for the Nifty 100 companies built using Python, SQLite, Streamlit, FastAPI, and Power BI.

---

## 📌 Project Overview

This project is designed to ingest, validate, analyze, and visualize financial data of Nifty 100 companies.

The platform includes:

- ETL Pipeline
- Data Quality Validation
- SQLite Database
- Financial Ratio Analysis
- Stock Price Analytics
- Streamlit Dashboard
- FastAPI Backend
- Automated Reports
